###UNIVERSAL SOUNDPACK SOLARIS URBINO###
Representing Cummins ISB and DAF PR

###Deutsch###

Installation:

Als erstes den alten Soundordner des jeweiligen Busses löschen. Dann extrahiert ihr den sound-ordner in dieser zip-Datei in den Hauptordner des Busses. Dann extrahiert ihr den Inhalt des jewiligen Solaris-Ordners ebenfalls in den Hauptordner.

Zuletzt die [sound] und [sound_AI] Einträge in der .bus-Datei ändern:
Für Cummins mit Voith D854:
sound\Solaris_Cummins_Voith.cfg
Für DAF mit Voith D854:
sound\Solaris_DAF_Voith.cfg
Für DAF mit ZF 6HP
sound\Solaris_DAF_ZF.cfg

Wichtig: Damit der ZF-Sound funktioniert, benötigt ihr das Script von Krtz07 hier zu finden: http://www.omnibussimulator.de/forum/index.php?page=Thread&threadID=10657
Nur die Scriptdateien werden benötigt und muss selbst installiert werden.

###English###

Installation:

First delete the old sound folder. Then extract the sound folder in this zip-file into the main folder of the bus you wish to use with this sound. Next extract the contents of the corresponding Solaris-folder also into the main folder. It contains the sound.cfg's and a doorscript with new door speeds.

Last thing to do is to change the [sound] and [sound_ai] entries to:
For Cummins with Voith D854:
sound\Solaris_Cummins_Voith.cfg
For DAF with Voith D854:
sound\Solaris_DAF_Voith.cfg
For DAF with ZF 6HP
sound\Solaris_DAF_ZF.cfg

Important: For the ZF-sound to work, you'll need Krtz07's script found here: http://www.omnibussimulator.de/forum/index.php?page=Thread&threadID=10657
Only the script files are needed and have to be manually installed.

Special thanks to Krtz07 for his ZF-sounds and great ZF script.
