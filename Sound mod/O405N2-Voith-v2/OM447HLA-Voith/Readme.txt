Installation:

Einfach den Sound-Ordner in den Busordner eurer Wahl entpacken. Als nächstes sucht ihr in der .Bus-Datei des jeweiligen Busses nach folgender Zeile:
[sound] und [sound_ai]
dort fügt ihr folgendes ein:
sound\sound_om447hla_voith.cfg oder
sound\sound_om447hla_voith_leise.cfg für eine realere Geräuschkulisse vorne im Cockpit.

Empfohlen wird des 4-stufige Voithskript von PoweredByCNG, zu finden bei Busmods im Forum.

Optional (und auch empfohlen):

Im Script-Ordner befindet sich eine Datei, welche den Kompressor häufiger laufen lässt, aber dafür nicht so lange. Ideal für den Kompressor des O405N². Vorher bitte eine Kopie von "bremse.osc" machen.

ENGLISH

Installation:

Just put the sound-folder into the bus-folder  you want to use with that soundpack. Then you have to edit the .bus-file and search for following entry:
[sound] and [sound_ai]
now substitute the existing entries with the following:
sound\sound_om447hla_voith.cfg or
sound\sound_om447hla_voith_leise.cfg for a more real representation of the sound in the cockpit. Whatever you like ;)

I recommend you to use PoweredByCNG's 4-stepped Voith gearbox-script. It can be found in the Busmods section inside the official forum.

Optional (and also recommended):

The script-folder contains a file which changes the behaviour of the air compressor. It will now run more often but not so long like before. Ideal for the O405N²'s compressor. Please make a backup of "bremse.osc"
